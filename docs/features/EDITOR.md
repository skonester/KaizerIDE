# Editor

KaizerIDE uses Monaco Editor to provide a familiar, capable code editing experience inside a lightweight desktop app.

[← Back to features](../FEATURES.md) · [Docs hub](../README.md)

---

## Monaco editor

KaizerIDE is built around Monaco, the same editor core used by VS Code. This gives the editor a familiar feel for navigation, selection, editing, and language-aware features.

---

## Editing features

- **Syntax highlighting** - Support for many common programming languages.
- **IntelliSense** - Auto-completion and intelligent suggestions where language support is available.
- **Multi-tab editing** - Open and work across multiple files at once.
- **Minimap** - Quickly navigate large files.
- **Auto-save** - Keep work protected during active editing.
- **Bracket colorization** - Improve readability in nested code.

---

## Themes

KaizerIDE supports customizable editor themes, including project-specific dark theme directions such as Kaizer Dark and Zero Syntax.

Appearance and editor options can be adjusted from Settings.

---

## Common actions

| Shortcut | Action |
| --- | --- |
| `Ctrl+S` | Save current file |
| `Ctrl+W` | Close current tab |
| `Ctrl+Shift+P` | Open command palette |
| `Ctrl+,` | Open settings |
