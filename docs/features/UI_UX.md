# UI & UX

KaizerIDE focuses on a polished desktop experience with familiar navigation, configurable appearance, and fast interaction patterns.

[← Back to features](../FEATURES.md) · [Docs hub](../README.md)

---

## Welcome screen

The welcome screen gives quick access to recent projects, folders, and the first actions needed to start coding.

---

## Command palette

`Ctrl+Shift+P` opens a fuzzy command launcher for keyboard-first workflows.

The command palette provides quick access to actions such as:

- **New file**.
- **Save**.
- **Toggle sidebar**.
- **Open settings**.
- **Re-index workspace**.
- **Connect through SSH**.

Navigation supports arrow keys and Enter for fast command execution.

---

## Appearance controls

Settings → Appearance includes controls for:

- **Window controls theme** - macOS-style or Windows-style controls.
- **Accent color** - Tune the app accent direction.
- **Compact mode** - Reduce spacing for denser layouts.
- **Status bar visibility** - Show or hide status details.

---

## Visual design

KaizerIDE uses a modern dark UI with glassmorphism-inspired surfaces, smooth animations, and native-feeling desktop chrome.

---

## Lazy-loaded modals

Modal-heavy surfaces are lazy-loaded to reduce initial bundle size and improve startup responsiveness.
