# Test Indexing Project

This is a test folder to verify the workspace indexer functionality.

## Files Included

- **example.js** - JavaScript with functions and classes
- **example.py** - Python with functions and classes
- **example.cpp** - C++ with classes and methods
- **example.lua** - Lua with functions and tables
- **config.json** - JSON configuration file
- **styles.css** - CSS stylesheet
- **README.md** - This file

## Purpose

Test that the indexer correctly:
1. Finds all code files
2. Extracts symbols (functions, classes)
3. Ignores binary files
4. Provides accurate file counts and structure
